# UPI Transaction Analytics Python-Sql

## Overview
This project simulates a real-world UPI transaction analytics and fraud detection system using Python and MySQL.

The project analyzes digital payment transactions to identify:
- Fraudulent transactions
- Failed payments
- Customer spending behavior
- Merchant performance
- Transaction success rates

---

## Technologies Used
- Python
- Pandas
- MySQL
- Jupyter Notebook

---

## Dataset
- 1000+ simulated UPI transaction records
- Includes:
  - User information
  - Transaction amount
  - Merchant details
  - Transaction status
  - Fraud risk indicators

---

## SQL Concepts Used
- Joins
- Aggregations
- GROUP BY
- ORDER BY
- CASE Statements
- Filtering

---

## Python Concepts Used
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Pandas DataFrames
- Fraud Detection Logic
- KPI Analysis

---

## Key Business Insights
- Highest spending users
- Most used merchants
- Transaction success vs failure rate
- Fraudulent transaction detection
- Customer transaction patterns
